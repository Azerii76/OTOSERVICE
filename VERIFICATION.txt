## HALA HATA ALIYORSANIZ:

### Sorun: Cache Sorunu
Android Studio eski dosyaları cache'lemiş olabilir.

### Çözüm:
1. File > Invalidate Caches > Invalidate and Restart
2. Projeyi kapatın
3. .gradle ve .idea klasörlerini silin
4. Projeyi yeniden açın
5. Build > Clean Project
6. Build > Rebuild Project

### Çözüm:

4. YENİ klasörü Android Studio'da açın
